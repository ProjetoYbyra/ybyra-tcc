# Ybyrá  
## RUN THE PROJECT <br/>
1. inicie o MySQL na máquina e crie o banco de dados à partir do documento src/components/database/database.sql
2. npm install 
3. npm install concurrently -g 
4. /ybyra-tcc/ - npm run projeto 


## RUN THE BUILD OF THE PROJECT
1. npm run build
2. npm run bp
